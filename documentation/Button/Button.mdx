## API

| 属性    | 说明                        | 类型    | 默认值  |
| ------- | --------------------------- | ------- | ------- | ------ | --------- |
| type    | 设置按钮类型                | default \| primary \| danger | default |
| size    | 设置按钮大小                | small   \| middle  \| large  | middle    |
| loading | 设置按钮是否为 loading 状态 | boolean | false   |
